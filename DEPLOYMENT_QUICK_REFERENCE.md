## 🔑 Environment Variables

### Render (Backend) - 5 Variables

```env
NODE_ENV=production
PORT=10000
MONGODB_URI=[Your MongoDB connection string]
JWT_SECRET=teacher-attendance-secret-2026-anuruddha-balika-secure-jwt-key
FRONTEND_URL=[Leave empty initially, add after Step 3]
```

### Vercel (Frontend) - 1 Variable

```env
VITE_API_URL=[Your Render backend URL]/api
```

**Example**:
```
VITE_API_URL=https://teacher-attendance-api.onrender.com/api
```

## 🐛 Common Mistakes to Avoid

| ❌ Wrong | ✅ Correct |
|----------|------------|
| `VITE_API_URL=https://backend.com` | `VITE_API_URL=https://backend.com/api` |
| `FRONTEND_URL=https://site.com/` | `FRONTEND_URL=https://site.com` |
| Deploy Frontend first | Deploy Database → Backend → Frontend |
| IP: 0.0.0.0 | IP: 0.0.0.0/0 (with /0) |
| Root: /backend | Root: backend (no slash) |

## 🚨 Emergency Fixes

### Backend Not Responding
```bash
# Wait 60 seconds (waking up from sleep)
# Check logs in Render dashboard
# Verify MONGODB_URI is correct
```

### CORS Errors
```bash
# Go to Render → Environment
# Check FRONTEND_URL has no trailing slash
# Wait 2 minutes for redeploy
# Hard refresh browser (Ctrl+Shift+R)
```

### Can't Login
```bash
# Go to MongoDB Atlas → Browse Collections
# Verify "users" collection exists
# Verify admin user document exists
# Check password hash is correct
```

### Build Failed
```bash
# Check logs for specific error
# Verify Root Directory is correct
# Check package.json exists
# Try manual redeploy
```
