# How to Use the New Navigation System 🎯

## Quick Start Guide

Your site now has a professional navigation system that makes it easy to move between pages and access all features.

---

## 🎓 Navigation Bar (Top of Every Page)

### What You'll See

```
┌─────────────────────────────────────────────────────────────┐
│ 🎓 Teacher Attendance  [Dashboard] [Attendance] [Substitutions] [Timetable] [Teachers]  👤 admin (admin) ▼ │
└─────────────────────────────────────────────────────────────┘
```

### How to Use

1. **Click any link** to go to that page instantly
2. **Current page** is highlighted with a purple background
3. **Admin-only features** show a 🔒 lock icon if you're not admin
4. **Click your name** (top right) to open user menu

---

## 📍 Breadcrumbs (Below Navigation Bar)

### What You'll See

```
Home › Attendance
```

### How to Use

1. **Click "Home"** to go back to dashboard
2. **Current page** is shown in gray (not clickable)
3. **Breadcrumbs only show** on pages other than dashboard

---

## 👤 User Menu (Top Right)

### What You'll See

```
┌─────────────────┐
│ admin           │
│ admin           │
├─────────────────┤
│ ⚙️ Settings     │
│ ❓ Help         │
├─────────────────┤
│ 🚪 Logout       │
└─────────────────┘
```

### How to Use

1. **Click your name** in top right to open menu
2. **Click "Logout"** to log out
3. **Settings and Help** coming soon!

---

## 📱 Mobile Menu (Small Screens)

### What You'll See

```
┌─────────────────────────────────┐
│ 🎓 Teacher Attendance      ☰   │
└─────────────────────────────────┘
```

### How to Use

1. **Tap the hamburger icon** (☰) to open menu
2. **Tap any link** to navigate
3. **Menu closes automatically** after navigation

---

## 🚀 Quick Navigation Examples

### Example 1: Go from Dashboard to Attendance
```
1. You're on Dashboard
2. Click "Attendance" in navigation bar
3. You're now on Attendance page
```
**Time**: 1 click, instant

### Example 2: Go from Attendance to Timetable
```
1. You're on Attendance page
2. Click "Timetable" in navigation bar
3. You're now on Timetable page
```
**Time**: 1 click, instant (no need to go back to dashboard!)

### Example 3: Go back to Dashboard
```
1. You're on any page
2. Click "Dashboard" in navigation bar
   OR
   Click "Home" in breadcrumbs
3. You're now on Dashboard
```
**Time**: 1 click, instant

### Example 4: Logout
```
1. You're on any page
2. Click your name in top right
3. Click "Logout" in dropdown
4. You're now on Login page
```
**Time**: 2 clicks, instant

## ⌨️ Keyboard Navigation

### Tab Through Links
1. Press **Tab** to move to next link
2. Press **Shift + Tab** to move to previous link
3. Press **Enter** to click current link

### Quick Access
1. Press **Tab** until you reach navigation
2. Use **Arrow keys** to move between links
3. Press **Enter** to navigate

## 🎯 Tips & Tricks

### Tip 1: Fastest Navigation
**Use the navigation bar** - it's always visible and always accessible. No need to scroll or go back to dashboard.

### Tip 2: Know Where You Are
**Look at the active state** in navigation bar and **breadcrumbs** to see your current location.

### Tip 3: Quick Logout
**Click your name** in top right, then **click "Logout"**. Works from any page!

### Tip 4: Mobile Navigation
**Tap the hamburger menu** (☰) to access all pages on mobile. Menu closes automatically after navigation.

### Tip 5: Keyboard Shortcuts
**Use Tab and Enter** to navigate without a mouse. Great for power users!

---

## 🔧 Troubleshooting

### Problem: Can't see navigation bar
**Solution**: Refresh the page (Cmd/Ctrl + R)

### Problem: Navigation bar looks weird
**Solution**: Clear browser cache and refresh

### Problem: Can't access a feature
**Solution**: Check if it's admin-only (🔒 icon). Ask admin for access.

### Problem: Mobile menu won't open
**Solution**: Tap the hamburger icon (☰) in top right

### Problem: Logout doesn't work
**Solution**: Click your name first, then click "Logout" in dropdown

## 📱 Mobile vs Desktop

### Desktop (Wide Screen)
- Navigation links always visible
- User menu in top right
- Breadcrumbs below navigation
- Footer at bottom

### Mobile (Small Screen)
- Hamburger menu (☰) in top right
- Tap to open full-screen menu
- Large touch targets
- Same features, different layout
